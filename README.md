项目介绍：用户可以注册和登录，注册的数据将写入文件供登录使用，登录后可以选择多种游戏，并且在游戏中创造的最高纪录将保存为对应账号的数据，该项目是游戏类项目与管理系统的结合，既是对入门级游戏的扩展也是对管理系统的创新。
